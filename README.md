# BlockFest

## Running the Project

This is a Solidity/web project. You can run it in several ways:

### Open in Browser Directly
Open `pages/index.html` in your browser (double-click or right-click → Open with browser)

### Run a Local Dev Server
```bash
npx serve
# or
npx http-server
```

### Compile Solidity Contracts
```bash
npx solc contracts/BlockFest.sol
```
