How to get started

  1. Install MetaMask Add the MetaMask browser extension from metamask.io and create or import a wallet.
  2. Get Sepolia ETH Use a Sepolia faucet to get free testnet ETH — you'll need it to pay for tickets and gas.
  3. Create or Join a Group Create a group with your friends or join an existing one using a Group ID.
